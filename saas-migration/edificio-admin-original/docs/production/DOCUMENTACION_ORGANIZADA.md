## 🗑️ Limpieza Recomendada

### Archivos que pueden eliminarse:
```bash
# Documentos obsoletos
rm DIAGNOSTICO_LOGIN.md
rm ESTADO_FINAL_CORRECCION.md
rm CORRECCION_RUTAS_FRONTEND.md

# Backups duplicados
rm -rf root/
rm -rf respaldo/EdificioDirs/

# Archivos .backup
find . -name "*.backup*" -type f
```

### Mantener por historial:
- `docs/reports/*` - Mantener todo el historial de reportes
- `backups/data-backups/*` - Mantener backups de datos
- Archivos `RESUMEN_*.txt` - Pueden mantenerse como referencia rápida
